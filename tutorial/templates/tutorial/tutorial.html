Rules of Tapa

Tapa is a puzzle where you use number hints to draw a single complete path through the grid until all the hints are satisfied. Puzzles in this tutorial are interactive. Left-click to blacken a tile and right-click to make the tile grey, marking it as not a part of the solution.

  1. In the first example the number 8 is the given hint.

  2. Hints determine how many of the surrounding tiles are colored in. So in this puzzle the 5 tiles around the "5" hint are filled in and so on. Hints which are as big as the number of adjecent tiles are especially powerful for this reason.

  3. If a hint uses more than one number then there must be a white tile between the blackened tiles. For the hint "2 2", there must be 2 sets of 2 tiles, with a white tile between them.

    4. *Dont forget that the black tiles must form one continuous path across the puzzle.


  4. For this puzzle the two corner tiles are white because of the last rule, that at no time can there be a 2x2 square of blackened tiles.
